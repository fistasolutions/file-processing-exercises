# Sales Analytics Project

A data analytics service for tracking sales, customers, and revenue.

## Quick Start

1. Copy `.env.local` values to your environment
2. Install dependencies: `pip install -r requirements.txt`
3. Run ETL pipeline: `python scripts/etl_pipeline.py`
4. Generate reports: `./scripts/generate_report.sh`

## Structure

- `data/` - Raw and processed data files
- `scripts/` - ETL and reporting scripts
- `reports/` - Generated reports
- `config/` - Configuration files
