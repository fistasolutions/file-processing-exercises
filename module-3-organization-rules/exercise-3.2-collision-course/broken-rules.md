# File Organization Rules

## Rule 1: Documents

Move all `.PDF` files to the `Documents/` folder.

## Rule 2: Invoices

Move all files with "invoice" in the name to the `Invoices/` folder.

## Rule 3: Images

Move all `.png` and `.jpg` files to the `Images/` folder.

## Rule 4: Contracts

Move all `.docx` files to the `Documents/` folder.
Move all files with "contract" or "agreement" or "NDA" in the name to the `Legal/` folder.

## Rule 5: Code Files

Move all `.py`, `.js`, and `.sh` files to the `Code/` folder.

## Rule 6: Spreadsheets

Move all `.csv` and `.xlsx` files to the `Spreadsheets/` folder.

## Rule 7: Text Files

Move all `.txt` and `.md` files to the `Notes/` folder.

## Rule 8: Design Files

Move all `.psd` files to the `Design/` folder.
